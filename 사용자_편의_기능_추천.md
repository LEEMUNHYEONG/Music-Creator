# Music Creator — 사용자 편의 추가 기능 추천

현재 6단계 워크플로우(가사 작성 → 수노 변환 → AI 분석 → 개선안 → 최종 출력 → 마케팅)를 기준으로, 사용자 편의를 높일 수 있는 기능을 우선순위별로 정리했습니다.

---

## 1. 바로 적용하기 좋은 기능 (우선순위 높음)

### 1.1 키보드 단축키
| 단축키 | 동작 | 효과 |
|--------|------|------|
| **Ctrl+S** (Mac: Cmd+S) | 현재 프로젝트 저장 | 자주 쓰는 저장을 빠르게 실행 |
| **Ctrl+1 ~ 6** | 해당 단계로 이동 | 마우스 없이 단계 전환 |
| **Esc** | 열린 모달/사이드바 닫기 | 모달 닫기를 일관되게 |
| **Ctrl+Enter** | 현재 단계의 "다음 단계로" 실행 | 워크플로우 속도 향상 |

- 구현: `document.addEventListener('keydown', ...)` 로 전역 키 이벤트 처리, 모달/사이드바는 `Esc`로 닫기.

### 1.2 저장 피드백 개선 (알림 → 토스트)
- **현재**: `alert()` 로 저장 완료/실패 표시.
- **개선**: 화면 한쪽에 잠깐 나타났다 사라지는 **토스트 메시지** (예: "저장되었습니다", "저장 실패").
- **효과**: 작업 흐름이 끊기지 않고, 반복 작업 시 피로 감소.

### 1.3 "Suno용 한 번에 복사"
- **위치**: 5단계(최종 출력) 또는 2단계(수노 변환) 결과 영역.
- **동작**: "가사 + 스타일 프롬프트"를 Suno 입력 형식으로 정리해 **한 번에 클립보드에 복사**.
- **효과**: Suno 사이트에 붙여넣기만 하면 되어 단계 수 감소.

### 1.4 프로젝트 복제
- **위치**: 사이드바 프로젝트 목록 각 항목 또는 상세 보기.
- **동작**: "복제" 클릭 시 같은 내용으로 **새 프로젝트 생성** (제목에 "복사본" 등 표시).
- **효과**: 비슷한 곡을 여러 버전으로 만들 때 1단계부터 다시 입력할 필요 감소.

### 1.5 최근 프로젝트 빠른 접근
- **위치**: 헤더 또는 사이드바 상단.
- **동작**: "최근 3~5개" 프로젝트만 버튼/링크로 표시, 클릭 시 해당 프로젝트 로드.
- **효과**: 자주 쓰는 프로젝트를 목록 검색 없이 열기.

## 2. 워크플로우 보완 (우선순위 중간)

### 2.1 단계별 진행률 표시
- **표시**: 예) "2/6 단계 완료", 또는 단계 아이콘에 체크 표시.
- **기준**: 해당 단계에 필수 입력(가사, 스타일 등)이 있으면 "완료"로 간주.
- **효과**: 지금 어디까지 했는지 한눈에 파악, 이어서 하기 쉬움.

### 2.2 자동 저장(드래프트)
- **동작**: 입력 중인 내용을 일정 간격(예: 30초~1분)마다 **임시 저장** (localStorage 또는 별도 키).
- **표시**: "마지막 자동 저장: 14:32" 같은 짧은 문구.
- **효과**: 새로고침/탭 닫기 실수 시 손실 감소 (선택 시에만 켜기 권장).

### 2.3 스타일 프롬프트 프리셋
- **위치**: 1단계 또는 2단계 스타일 입력란 근처.
- **동작**: "발라드", "팝 록", "R&B" 등 **미리 정의된 스타일** 버튼 선택 시 해당 문구가 입력란에 채워짐.
- **효과**: 매번 긴 문장을 치지 않아도 되고, 품질 일관성 유지에 도움.

### 2.4 가사 템플릿(구조)
- **동작**: "Verse–Chorus–Bridge" 등 **구조만 있는 빈 템플릿** 선택 시 가사 입력란에 섹션 이름만 채워 넣기.
- **효과**: 처음 쓰는 사람도 구조를 따라가기 쉬움.

### 2.5 단계별 예상 소요 시간
- **표시**: 각 단계 카드에 "보통 약 2분" 같은 안내 문구.
- **효과**: 전체 작업 시간 예측과 계획에 도움.

## 3. 접근성·환경 설정 (우선순위 중간)

### 3.1 다크/라이트 테마 전환
- **위치**: 헤더 "설정" 또는 전용 토글.
- **동작**: 사용자 선택을 localStorage에 저장해 다음 방문 시에도 유지.
- **효과**: 장시간 사용 시 눈 피로 완화, 선호도 반영.

### 3.2 글자 크기 조절
- **동작**: "작게 / 보통 / 크게" 3단계 정도로 본문/입력란 글자 크기만 조절.
- **효과**: 가독성 개인화.

### 3.3 모달 키보드 접근
- **동작**: 모달이 열리면 포커스를 모달 안으로 이동, **Tab**으로만 모달 안 요소들 순환, **Esc**로 닫기.
- **효과**: 키보드만으로도 모든 기능 사용 가능.

### 3.4 "맨 위로" 버튼
- **위치**: 긴 단계(가사, 마케팅 등) 스크롤 시 화면 한쪽에 "맨 위로" 버튼 표시.
- **효과**: 긴 페이지에서 빠르게 상단으로 이동.

---

## 4. 데이터·백업 (우선순위 중~낮음)

### 4.1 단일 프로젝트 내보내기/가져오기
- **내보내기**: 프로젝트 하나를 **JSON 파일**로 다운로드.
- **가져오기**: JSON 파일 선택 시 해당 프로젝트를 목록에 추가 및 로드.
- **효과**: 백업·다른 기기로 옮기기·공유가 쉬움.

### 4.2 프로젝트 태그/폴더(카테고리)
- **동작**: 프로젝트에 "발라드", "작업중", "완료" 등 **태그** 부여, 사이드바에서 태그별 필터.
- **효과**: 프로젝트 수가 많아질 때 정리·검색 용이.

### 4.3 삭제 전 확인 + 휴지통
- **동작**: 삭제 시 "정말 삭제할까요?" 확인 후, **일정 기간(예: 7일) 휴지통**에 보관했다가 완전 삭제.
- **효과**: 실수로 삭제한 프로젝트 복구 가능.

---

## 5. AI·콘텐츠 보조 (선택)

### 5.1 AI 분석 진행률 표시
- **동작**: 3단계 "Gemini 정밀 분석" 등 **로딩 중 진행률** 또는 단계별 메시지(예: "가사 분석 중…" → "개선안 생성 중…").
- **효과**: 대기 시간에 대한 불안 감소.

### 5.2 이전 생성 결과 재사용
- **동작**: 가사/AI 분석 결과 등에 "이 결과로 다시 변환" 또는 "이 개선안만 다시 적용" 버튼 제공.
- **효과**: 같은 입력으로 설정만 바꿔서 여러 번 시도하기 쉬움.

### 5.3 해시태그/썸네일 문구 즐겨찾기
- **동작**: 마음에 드는 해시태그·썸네일 문구를 "즐겨찾기"로 저장해 다음 프로젝트에서 빠르게 붙여넣기.
- **효과**: 자주 쓰는 문구 재입력 감소.

| 순서 | 기능 | 예상 공수 | 사용자 체감 |

| 1 | Ctrl+S 저장 단축키 | 낮음 | 높음 |

| 3 | Suno용 한 번에 복사 | 낮음 | 높음 |
| 4 | 프로젝트 복제 | 중간 | 높음 |
| 5 | Esc로 모달/사이드바 닫기 | 낮음 | 중간 |
| 6 | 단계 이동 Ctrl+1~6 | 낮음 | 중간 |
| 7 | 최근 프로젝트 3~5개 | 중간 | 중간 |
| 8 | 단계별 진행률(체크) | 중간 | 중간 |
| 9 | 다크/라이트 테마 | 중간 | 중간 |
| 10 | 스타일 프리셋 | 중간 | 중간 |

## 7. 정리

- **바로 넣기 좋은 것**: 키보드 단축키(Ctrl+S, Esc, Ctrl+1~6), 토스트 알림, Suno 한 번에 복사, 프로젝트 복제.
- **조금만 더**: 진행률 표시, 자동 저장(드래프트), 스타일 프리셋, 테마 전환.
- **나중에**: 태그/폴더, 휴지통, 단일 프로젝트 내보내기/가져오기.

원하시는 항목부터 단계적으로 적용하시면, 사용자 편의가 크게 올라갈 것입니다.
